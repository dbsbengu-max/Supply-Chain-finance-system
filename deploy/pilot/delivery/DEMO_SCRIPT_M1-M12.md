# 演示脚本 — M1 至 M12（约 45–60 分钟）

**适用**：Mock/试点环境、业务验收、对外演示  
**主账号**：`platform_admin / Admin@123`  
**灌数（可选）**：`apply-seed-profile.ps1 -Profile demo`

## 开场（2 分钟）

> 本次演示基于 V1.1 试点闭环，签章与银行为系统内 Mock/Adapter，不代表真实生产联调结果。真实 vendor Go/No-Go 已登记为后置待办。
