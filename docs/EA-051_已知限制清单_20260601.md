# EA-051 已知限制清单

**日期**：2026-06-01  
**版本**：V1.1.0（Mock/试点交付）  
**原则**：以下限制 **不阻断** Mock/试点演示与 EA-051 交付；真实生产前须逐项关闭或签核豁免。

---

## 1. 集成与外部依赖

| ID | 限制 | 影响 | Workaround | 关闭条件 |
|---|---|---|---|---|
| LIM-INT-01 | 签章为 Mock / HTTP Adapter，非真实 vendor | 无法证明生产 SLA/证书/UKey | 本地 quasi-sandbox 脚本；补偿池人工复核 | EA-048 GO |
| LIM-INT-02 | 银行通道为模拟 | 无真实来账/对账 | Mock 流水 + 清分试算演示 | 银行联调包 |
| LIM-INT-03 | 无生产 ALLOWLIST 灰度 | 不能按项目百分比切真实签章 | rollout 只读视图 | EA-047 + EA-048 |
| LIM-INT-04 | 回调 URL 为 localhost/内网 | 外网 vendor 无法主动回调 | 脚本 replay / stub | Staging 公网 URL |

## 2. 环境与运维

| ID | 限制 | 影响 | Workaround | 关闭条件 |
|---|---|---|---|---|
| LIM-OPS-01 | 依赖本机 Postgres + 手工启停后端 | 无高可用 | `start-backend-local.ps1` | 运维 Staging/Pilot |
| LIM-OPS-02 | 默认 dev 密码 | 安全风险若暴露公网 | 仅内网 UAT；生产轮换 | EA-033 加固 |
| LIM-OPS-03 | 无 24h 监控/告警/回滚演练 | 生产故障发现慢 | 试点阶段人工巡检 | P-06 |
| LIM-OPS-04 | Flyway PG16 版本警告 | 迁移告警噪音 | 可忽略或降 PG 版本 | 运维基线 |
