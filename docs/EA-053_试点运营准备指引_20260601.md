## 5. P0 判定（可开 hotfix）

- 无法登录 / 8080 或 5173 不可用（非环境未启动）
- 主链路 403 误拦 `platform_admin`
- `run-ea051-acceptance.ps1` 或 smoke 9/9 失败（非环境变量污染）
- 补偿池 / 签章 Mock 导致演示完全中断

**非 P0** → 记入 [`USER_FEEDBACK_LOG.md`](../deploy/pilot/ops/USER_FEEDBACK_LOG.md)，排期 backlog，**不开** pilot hotfix PR。
