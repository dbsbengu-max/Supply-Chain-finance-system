# EA-034 Staging 执行包与验收

生成日期：2026-05-31（更新：真实环境闸门 + seed 拆分）

## 1. 目的

Staging **真实环境闸门**：seed 归档、StrictStale 连续告警、IAM/demo seed 拆分评估与可选应用。

## 2. 闸门一键执行（推荐）

```powershell
cd deploy\pilot
copy .env.staging.example .env   # 编辑 staging 连接
.\scripts\run-staging-gate.ps1 -WatchMinutes 30 -WatchIntervalMinutes 5
```

产出：

- `evidence/staging/seed-verify-*.log`
- `evidence/staging/alerts-watch-*.log`
- `evidence/staging/staging-gate-*.summary.md`

仅 seed + 单次 StrictStale（跳过 30min watch）：

```powershell
.\scripts\run-staging-gate.ps1 -SkipAlertsWatch
```

## 3. 分步命令

```powershell
.\scripts\verify-pilot-seed.ps1 -ArchiveDir .\evidence\staging
.\monitoring\check-pilot-alerts.ps1 -StrictStale -BackendUrl $env:SCF_API_HEALTH_URL
.\scripts\run-alerts-watch.ps1 -IntervalMinutes 5 -Iterations 12 -StrictStale -ArchiveDir .\evidence\staging
```
