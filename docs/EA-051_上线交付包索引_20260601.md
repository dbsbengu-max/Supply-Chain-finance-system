## 2. 一键验收（推荐）

**前提**：PostgreSQL `:5432` 可用；本机 Java 17（建议 D 盘 Temurin）、Maven 3.9+。

```powershell
cd "e:\cursor\BR PT\供应链金融管理系统\deploy\pilot\scripts"
.\run-ea051-acceptance.ps1
```

可选参数：

| 参数 | 说明 |
|---|---|
| `-SkipRegression` | 跳过后端 171/171 |
| `-SkipBuild` | 跳过 `npm run build` |
| `-SkipSmoke` | 跳过前端 smoke 9/9 |
| `-EnsureBackend` | 若 8080 未 UP 则调用 `start-backend-local.ps1` |
| `-ApplyDemoSeed` | 验收前执行 `apply-seed-profile.ps1 -Profile demo` |

> **Note**：若本机 shell 已设置 `SCF_CONTRACT_SIGN_CALLBACK_TOKEN`（如 dev 后端），脚本会在 A1 回归前自动重置为 `mock-contract-sign-callback-token`，避免 EA040 用例 403。

**期望输出**：末尾 `>>> EA-051 ACCEPTANCE: PASS <<<`
