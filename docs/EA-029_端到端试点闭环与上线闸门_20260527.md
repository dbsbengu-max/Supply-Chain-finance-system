## 4. 异常补偿演练（4 条）

| ID | 场景 | 步骤 | 预期 |
|---|---|---|---|
| CMP-01 | 补偿执行失败 | 错误 account_id 触发 FAILED | retry_count+1，next_retry_at 写入 |
| CMP-02 | 自动重试窗口 | 未到期再次 process | retry_count 不变 |
| CMP-03 | 达上限 MANUAL_REQUIRED | 连续失败至上限 | 状态 MANUAL_REQUIRED，审计 SAGA_COMPENSATION_MANUAL |
| CMP-04 | 人工修正后重试 | Saga 台填原因 → 人工重试 | SUCCESS，审计含 manual_reason |
