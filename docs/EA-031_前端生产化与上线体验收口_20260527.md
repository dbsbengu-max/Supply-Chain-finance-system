# EA-031 前端生产化与上线体验收口

生成日期：2026-05-27

## 1. 目标

降低首屏 JS 体积、按需加载 Element Plus、固化浏览器试点 smoke，并清理构建缓存跟踪。

## 2. Cursor 交付物

### 2.1 Element Plus 按需导入

- `unplugin-auto-import` + `unplugin-vue-components` + `ElementPlusResolver({ importStyle: 'css' })`
- `main.ts` 移除全量 `app.use(ElementPlus)`，仅注册 `v-loading` 指令（`ElLoading.directive`）
- 生成类型：`src/auto-imports.d.ts`、`src/components.d.ts`

### 2.2 路由懒加载 + manualChunks

- 路由全部改为 `() => import(...)`（含 Login、Layout、Workbench）
- `vite.config.ts` 拆分：`vue-vendor`、`element-plus`、`echarts`、`vendor`、`domain-*`

### 2.3 浏览器 Smoke

- Playwright：`tests/smoke/pilot-routes.spec.ts`（登录、试点闭环、Saga、融资、清分、BI）
- 一键执行：`npm run smoke`（输出 PASS/FAIL 汇总）
- 前置条件：后端 `http://localhost:8080` 已启动（Vite 代理 `/api/v1`）

```powershell
cd frontend/scf-web
npm run smoke:install   # 首次安装 Chromium
npm run smoke           # 需 backend + 自动 reuse dev server
# 或已有 dev 服务：$env:SMOKE_SKIP_WEBSERVER=1; npm run smoke
```

### 2.4 构建缓存卫生

- `.gitignore` 增加 `test-results/`、`playwright-report/`
- `tsconfig.tsbuildinfo` 已在 EA-030 停止跟踪；本地文件不入库

## 3. 包体对比（`npm run build`）

| 产物 | EA-030 基线（约） | EA-031 后 |
|---|---|---|
| 主入口 `index-*.js` | ~1,830 KB / gzip ~602 KB | **~10 KB / gzip ~3.3 KB** |
| `element-plus-*.js` | （合并在主包） | ~794 KB / gzip ~251 KB（独立 chunk） |
| `vue-vendor` | （合并） | ~122 KB / gzip ~47 KB |
| 业务域 chunk | 无 | `domain-pilot` / `domain-saga` / `domain-bi` / `domain-account` 按需加载 |

首屏仅加载入口 + 当前路由 chunk + 共享 vendor，显著低于单包 1.83MB。
