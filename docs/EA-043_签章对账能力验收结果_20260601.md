## Smoke 场景（自动化）

1. 发起签署 → `SIGNING`
2. 成功回调 → `SIGNED`
3. 失败回调 → `FAILED`
4. 未知 `external_sign_ref` 回调 → 404 + 补偿池 `MANUAL_REQUIRED`
5. Saga 主动查单（无本地任务）→ `provider_status=UNKNOWN`，`reconciled=false`
6. 忽略补偿 → `IGNORED`
7. 待回调任务 + Mock 供应商 SUCCESS → 查单对账 → `SIGNED`
8. 未知回调补偿关闭 → `CLOSED`

## 权限

- 查单接口：`CONTRACT_SIGN_STATUS_QUERY`
- Saga 查单：另需 `SAGA_OPS_HANDLE` 或 `SAGA_OPS_MANAGE`
- 成员角色无 `CONTRACT_SIGN_STATUS_QUERY` → `403 AUTH_403`
