## 5. Agent 工作流（Cursor / Codex）

### 5.1 每次改动前

1. 读 `docs/EA-051_上线交付包索引_20260601.md`  
2. 确认需求属于 **白名单**  
3. 若触及 DEF-048 / BLOCKER-P → **停止**，只更新文档

### 5.2 每次改动后（必须）

```powershell
cd deploy\pilot\scripts
.\run-ea051-acceptance.ps1 -SkipRegression   # 仅前端时可
# 或完整：
.\run-ea051-acceptance.ps1
```

| 门禁 | 阈值 |
|---|---|
| A1 回归 | 171/171 |
| A2 build | PASS |
| A3 smoke | 9/9 |
| A4 health | UP |

### 5.3 提交信息约定

```
fix(ea-051): <what> — bugfix, regression/smoke PASS
docs(ea-051): <what> — no code behavior change
chore(ea-048-deferred): <what> — docs/scripts only
```

### 5.4 禁止的 Agent 行为

- 用 local quasi-sandbox PASS 宣称 EA-048 GO  
- 默认开启 `SCF_CONTRACT_SIGN_ROLLOUT_MODE=ALLOWLIST`  
- 提交 `.env` 真实密钥  
- 为通过演示而 skip 权限测试
