# EA-037 Saga补偿闭环与人工处置台开发简报

日期：2026-05-31

## 1. 背景与决策

EA-035/EA-036 的 staging / 准 staging 环境验证暂缓，不阻塞后续功能开发；但它仍是上线前 Go/No-Go 的阻断项。

当前功能主链路已经覆盖：

- 代采申请、审批、Saga 主链路；
- 保证金冻结、库存冻结、融资创建；
- Outbox、补偿任务、Saga 监控台基础能力；
- 凭证融资、还款清分、凭证释放、兑付审批。

下一步不建议继续横向扩新模块，而应补齐“失败后如何运营处理”的纵向闭环。推荐推进 EA-037：**Saga补偿闭环与人工处置台**。

## 2. 跳过 staging / 准 staging 的风险

| 风险 | 影响 | 当前处理 |
|---|---|---|
| 环境差异未验证 | PostgreSQL、网络、配置、密钥、文件路径、时区等问题可能延后暴露 | 不影响开发；上线前必须补 EA-035 |
| Flyway真实库迁移未验证 | 可能存在本地 H2/测试库通过、真实 PostgreSQL 失败 | 开发阶段继续跑集成测试；上线前真实库跑闸 |
| 性能与资源未验证 | 大包体、慢SQL、启动内存、连接池问题无法确认 | 暂缓性能结论 |
| 运维脚本未真实闭环 | seed、watch、evidence、Go/No-Go 证据链不能签字 | 保持 No-Go |
| 安全配置未真实注入 | JWT、BANK token、DB密码、日志策略未被环境证明 | 不能发布生产 |

结论：**可以继续开发；不能宣称可上线。**

## 3. EA-037目标

把 Saga 失败后的系统行为从“记录失败”升级为“可运营处置”：

1. 订单确认后的回滚补偿策略可执行；
2. 补偿任务可人工认领、审批、重试、忽略、关闭；
3. Saga/Outbox/补偿任务详情能解释失败原因、影响范围和建议动作；
4. 形成可测试的异常闭环，支撑 UAT 和上线巡检。

## 4. 本期范围

### 4.1 后端范围

| 能力 | 说明 |
|---|---|
| `ORDER_ROLLBACK` 补偿动作 | 代采 Saga 在订单已确认、后续步骤失败时，可将订单回退到可解释状态 |
| 补偿任务动作扩展 | 支持 `CLAIM`、`RETRY`、`APPROVE_EXECUTE`、`IGNORE`、`CLOSE` |
| 人工处置状态机 | `PENDING` / `RETRYING` / `MANUAL_REQUIRED` / `CLAIMED` / `APPROVED` / `IGNORED` / `CLOSED` / `SUCCESS` / `FAILED` |
| 处置备注与审批 | 所有人工动作必须有 reason；高风险动作需四眼审批 |
| 影响范围摘要 | 返回订单、融资、库存、账户、凭证相关影响字段 |
| 审计日志 | 所有人工动作写入 `audit_log` |

### 4.2 前端范围

| 页面 | 说明 |
|---|---|
| Saga监控台增强 | 增加补偿任务列表、状态筛选、风险标识 |
| 补偿任务详情抽屉 | 展示失败步骤、错误、影响对象、建议动作、审计时间线 |
| 人工处置动作 | 认领、提交审批、审批执行、忽略、关闭、重试 |
| 代采详情联动 | 代采详情页展示补偿任务和处置入口 |

### 4.3 测试范围

| 用例 | 验收 |
|---|---|
| 订单确认成功后融资创建失败 | 生成 `ORDER_ROLLBACK` 补偿任务 |
| `ORDER_ROLLBACK` 执行成功 | 订单状态回退，Saga记录成功补偿 |
| 非授权用户重试补偿 | 403 |
| 人工忽略必须填写原因 | 缺 reason 返回 400 |
| 发起人不可自审高风险补偿 | 返回 `BPM_FOUR_EYES_409` |
| 重复执行补偿 | 幂等，不重复回滚/解冻 |
| 补偿关闭后不可重试 | 返回 409 |

## 6. Cursor任务简报

### 任务目标

实现 EA-037：Saga补偿闭环与人工处置台，优先补齐 `ORDER_ROLLBACK` 补偿动作、补偿任务人工处置状态机、前端处置入口和集成测试。

### 技术约束

- 不改历史 Flyway 文件，只新增 `V1_1_028__*.sql`。
- 不绕过现有 `PermissionService`、`DataScopeHelper`、审计日志。
- 高风险人工动作必须写审计；涉及资金/货权/订单回退必须幂等。
- 不把 staging / 准 staging 验证作为本任务前置。

### 参考文档

1. `docs/EA-027_代采跨域Saga主链路验收结果_20260531.md`
2. `docs/EA-028_补偿Worker与Saga监控台验收结果_20260527.md`
3. `docs/EA-035_Codex复核结果_20260531.md`

### 禁止事项

- 禁止删除或重写既有补偿任务记录。
- 禁止用前端隐藏替代后端鉴权。
- 禁止把人工忽略/关闭作为默认自动补偿路径。
- 禁止在无审计日志的情况下执行补偿。

### 验收标准

- 新增集成测试不少于 7 条，覆盖 §4.3。
- `EA019RegressionRunner` 纳入新增测试并通过。
- 前端构建通过。
- `docs/EA-037_Saga补偿闭环与人工处置台验收结果_20260531.md` 输出验收结论。
